## 🏦 JavaBank — Projeto em Destaque

> Sistema bancário completo construído do zero para consolidar OOP em Java.

```
✅ 4 tipos de conta com hierarquia de herança real
✅ 94 testes unitários JUnit 5 — TDD na prática
✅ Arquitetura MVC com separação de camadas
✅ Console interativo + Interface JavaFX em desenvolvimento
✅ Java 21: Streams, Optional, lambdas, pattern matching
```

**O que aprendi de verdade:**
- Por que usar classe abstrata em vez de interface (e vice-versa)
- Como testar antes de terminar o código e por que isso importa
- Que trocar a interface (console → JavaFX) sem tocar o domínio
  é o resultado de uma boa arquitetura, não de sorte
